# level-microblog

A simple micro-blogging library for leveldb/[levelup](https://github.com/rvagg/node-levelup).

[![build status](https://secure.travis-ci.org/eugeneware/level-microblog.png)](http://travis-ci.org/eugeneware/level-microblog)

# Installation

This module is installed via npm:

``` bash
$ npm install level-microblog
```
